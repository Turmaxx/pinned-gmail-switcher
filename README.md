# Pinned Gmail Switcher

Pinned Gmail Switcher is a lightweight Firefox extension that opens Gmail in a pinned tab or switches to an existing Gmail tab if one is already open.

It was created because there weren’t any simple extensions that provide a true one-click Gmail launcher without extra features or clutter.

## Privacy

No data is collected or transmitted.
